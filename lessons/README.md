# 📚 Lessons Library

This folder contains all RFree Academy lessons organized by course/track.

## Structure
```
/lessons/
  /Financial-Literacy/
    lesson-1.md
    lesson-2.md
  /Building-Your-First-Budget/

```

## Auto-Sync
Lessons are automatically synced from the RFree Academy app daily at 9 AM EST.

Each lesson includes:
- Title
- Content/Outline
- Metadata (duration, grade level, etc.)
